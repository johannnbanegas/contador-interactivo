# Contador Interactivo

Mi primer proyecto web con JavaScript puro.

## Características
- Incrementar y decrementar números
- Multiplicar por 2
- Resetear contador
- Cambio de colores según el valor
- Atajos de teclado

## Tecnologías
- HTML5
- CSS3
- JavaScript Vanilla
